# Custom Generated Matrices (Legacy Path)

`benchmarks/matrices/generated/` is now the recommended location for new synthetic datasets.
This `custom/` directory remains as a backward-compatible path.

This directory is the home for synthetic benchmark matrices generated by:

```bash
python3 benchmarks/scripts/generate_matrices.py \
  --spec benchmarks/matrices/custom/specs/example.json \
  --out benchmarks/matrices/custom \
  --force
```

Generator outputs:

- `raw/*.mtx`: generated MatrixMarket coordinate inputs
- `canonical/*.mtx`: canonicalized matrices used by benchmark runners
- `manifest.csv`: generator metadata + computed matrix characteristics
- `pairs.csv`: optional explicit pair list for two-sparse kernels

Run unified benchmarks against custom matrices:

```bash
python3 benchmarks/scripts/benchmark_all.py \
  --matrices-dir benchmarks/matrices/custom/raw \
  --canonical-dir benchmarks/matrices/custom/canonical \
  --matrix-manifest benchmarks/matrices/custom/manifest.csv \
  --max-matrices 10
```

For pair kernels with explicit pairs:

```bash
python3 benchmarks/scripts/benchmark_spgemm.py \
  --matrices-dir benchmarks/matrices/custom/raw \
  --canonical-dir benchmarks/matrices/custom/canonical \
  --matrix-manifest benchmarks/matrices/custom/manifest.csv \
  --pairs-file benchmarks/matrices/custom/pairs.csv \
  --max-pairs 10
```
